# Co dalej? -- Sciezki nauki

Zbudowales dwa projekty napedzane AI i poznales fundamenty. Oto gdzie mozesz pojsc glebiej.

---

## AI Engineering (buduj produkty)

**Frameworki:**
- [LangChain](https://python.langchain.com/) -- najpopularniejszy framework do apek AI. Dobry do RAG, chains, agentow.
- [LlamaIndex](https://www.llamaindex.ai/) -- skupiony na ingestii danych + RAG. Swietny do "czatuj ze swoimi dokumentami."
- [CrewAI](https://www.crewai.com/) -- orkiestracja wielu agentow. Buduj zespoly agentow AI.
- [Semantic Kernel](https://learn.microsoft.com/en-us/semantic-kernel/) -- SDK orkiestracji AI od Microsoftu.

**RAG (Retrieval-Augmented Generation):**
- Juz zbudowales podstawowy RAG z embeddingami + Qdrant w Projekcie 1!
- Nastepne kroki: strategie chunkowania, wyszukiwanie hybrydowe, reranking, multi-modalny RAG
- [Dokumentacja Qdrant](https://qdrant.tech/documentation/) -- wyjdz poza in-memory

**Deployment:**
- FastAPI + Docker + Azure/AWS na produkcyjne API
- [Modal](https://modal.com/) -- serverless GPU compute do AI workloadow
- [Replicate](https://replicate.com/) -- odpalaj open-source modele jako API

## Agenci (buduj autonomiczne systemy)

**Ekosystem MCP:**
- [Specyfikacja MCP](https://modelcontextprotocol.io/) -- pelna dokumentacja protokolu
- [Rejestr serwerow MCP](https://github.com/modelcontextprotocol/servers) -- istniejace serwery do uzycia
- Buduj wlasne: dostep do bazy danych, wewnetrzne API, systemy plikow

**Frameworki agentowe:**
- [AutoGen](https://microsoft.github.io/autogen/) -- framework multi-agentowy od Microsoftu
- [LangGraph](https://langchain-ai.github.io/langgraph/) -- stanowe workflow agentow z LangChain

## ML Engineering (trenuj i wdrazaj modele)

**Fine-tuning:**
- [Hugging Face PEFT](https://huggingface.co/docs/peft) -- LoRA, QLoRA, parameter-efficient fine-tuning
- [Unsloth](https://github.com/unslothai/unsloth) -- szybki fine-tuning, 2x przyspieszenie
- [Axolotl](https://github.com/OpenAccess-AI-Collective/axolotl) -- prosty tool do fine-tuningu

**MLOps:**
- [MLflow](https://mlflow.org/) -- sledzenie eksperymentow, rejestr modeli
- [Weights & Biases](https://wandb.ai/) -- sledzenie eksperymentow + wizualizacja
- [Hugging Face Hub](https://huggingface.co/) -- hosting i udostepnianie modeli
